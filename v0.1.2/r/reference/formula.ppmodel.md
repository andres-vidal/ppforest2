# Formula extractor for ppforest2 models.

Formula extractor for ppforest2 models.

## Usage

``` r
# S3 method for class 'ppmodel'
formula(x, ...)
```

## Arguments

- x:

  A `pptr` or `pprf` model.

- ...:

  Unused.

## Value

The formula the model was trained with, or \`NULL\` for matrix-interface
fits.
